## 🚀 Sobre o Projeto

O **VivaLivre** é uma solução full-stack desenvolvida para resolver um problema real de mobilidade urbana e saúde: a dificuldade de encontrar banheiros acessíveis e pontos de suporte em tempo real. 

Utilizando geolocalização e processamento de dados espaciais, o app oferece uma interface intuitiva para que usuários possam localizar, avaliar e navegar até os pontos de apoio mais próximos, garantindo autonomia e conforto no dia a dia.

## 🛠️ Arquitetura Técnica

Diferente de soluções convencionais, o VivaLivre foi construído com uma arquitetura independente e escalável:

- **Mobile:** Desenvolvido em **Flutter**, focado em performance nativa e uma UX fluida para Android e iOS.
- **Backend:** Uma API REST robusta construída em **Golang**, garantindo alta concorrência e baixo tempo de resposta.
- **Banco de Dados:** Utilização de **PostgreSQL** com a extensão **PostGIS** para cálculos matemáticos de geolocalização por proximidade (Radius Search).
- **Segurança:** Autenticação proprietária via **JWT (JSON Web Tokens)** e proteção de dados com hashing seguro.
